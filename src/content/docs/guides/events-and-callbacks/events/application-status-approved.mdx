The "Application Approved" event is a critical webhook triggered when an application is approved within Funnel Leasing. This event is essential for tracking and managing application statuses, providing necessary data for integrations to update records and proceed with further actions.

## Event Details

- **Event Type**: `application_approved`
- **Description**: This event is triggered when an application is approved, signaling that the applicant has met the required criteria.
- **Event Payload Example**:

```json
{
  "event_id": "evt_1OxyzKLt4dXK03v5qaZabcDE",
  "event_type": "application_approved",
  "created_at": 1681234567,
  "event_data": {
      "application_id": 3333,
      "renter_id": 3333,
      "approved_by": "leasing_agent_001",
      "community_id": 54321
  }
}
```

## Payload Fields

- **`event_id`**: A unique identifier for the event.
- **`event_type`**: The type of event (`application_approved`).
- **`created_at`**: A timestamp indicating when the event occurred.
- **`event_data`**: Contains additional data related to the event:
  - **`application_id`**: The ID of the approved application.
  - **`renter_id`**: The proprietary renter ID within Funnel.
  - **`approved_by`**: The identifier of the leasing agent or system that approved the application.
  - **`community_id`**: The ID of the community where the application was approved.

## Use Cases

This event is crucial for integrations that need to track and manage application approvals. For instance, a Property Management System (PMS) integration can utilize this event to:

- **Update Application Status**: Automatically update the status of the application to 'approved' within the system.
- **Trigger Next Steps**: Initiate subsequent processes such as lease preparation, deposit requests, or welcome emails to the approved applicant.
- **Data Synchronization**: Sync the approval data across multiple systems to maintain consistent and up-to-date records.
- **Notify Stakeholders**: Send notifications to relevant stakeholders, including leasing agents and property managers, about the approval.

By leveraging this event, integrations can streamline the approval process, improve communication with applicants and staff, and ensure data integrity within the Funnel Leasing ecosystem.
